
# Pear-to-Pear (p2p storage) — MVP

Идея проекта: p2p-сервис для хранения набора файлов на разных устройствах.  
В MVP — **линейная структура без папок**, базовые операции: **добавить / удалить / скачать**, интерфейс — **терминал**.  
Метаданные синхронизируются между устройствами, сами файлы хранятся на устройствах участников.

## Что лежит в репозитории

- `docs/` — **структура и материалы проекта** (диздок / описание MVP / заметки)  
  > Здесь зафиксировано, что именно мы реализуем и как это задумано (аналог того, что рисовали на доске).

- `examples/` — **пробный код / эксперименты**  
  > Небольшие тестовые штуки, чтобы “потрогать” gRPC/protobuf/файлы и т.п.

## Технологии (MVP)
- Сеть: gRPC  
- Протокол: Protobuf  
- БД: SQLite  
- Файлы/блоки: `std::filesystem`  
- CLI: CLI11  
- Тесты: gTest
